Network mode

Standalone broadcasts the Alphasound WiFi for car/portable use. Client joins an existing network — useful at home, and unlocks "Check for updates" below. Wrong client credentials fall back to AP mode automatically.

Update from GitHub

Pulls alphasound.apkovl.tar.gz directly from the latest release. Only works in client mode (needs internet).

Update from file

Pick a downloaded alphasound.apkovl.tar.gz. The previous version is kept as a backup and automatically restored if the new one fails to boot 3 times.

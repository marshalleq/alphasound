Features

Disable what you don't use to shave seconds off boot. Reboots after applying.

Independent of display — pick a DAC with no screen for behind-dash installs.

HDMI takes over the console.

Applies live (Pirate Audio only).

Re-wakes the moment music resumes.

Roon Bridge

Turns the device into a Roon-compatible audio endpoint. Requires client mode (so the device can reach your Roon Core), a one-time download from Roon's CDN, and a Roon Labs subscription on your Core. Roon Bridge and AirPlay both grab the audio device exclusively — only one will play at a time.

Network mode

Standalone broadcasts the Alphasound WiFi for car/portable use. Client joins an existing network — useful at home, and unlocks "Check for updates" below. Wrong client credentials fall back to AP mode automatically.

Update from GitHub

Pulls alphasound.apkovl.tar.gz directly from the latest release. Only works in client mode (needs internet).

Update from file

Pick a downloaded alphasound.apkovl.tar.gz (or the bare .tar if your OS auto-extracted the wrapper). The previous version is kept as a backup and automatically restored if the new one fails to boot 3 times.
